Getting started

To get started with a virtual mechanism, perform the following.
  1. Open a virtual model from the Virtual Controls Laboratory landing page (e.g., RotaryPendulum.slx).
  2. ▶ Run the Simulink model.
  3. Place the Mechanics Explorer window next to the Simulink window.
  4. Hide the Tree View Pane by clicking the small black arrow ◂.
  5. Select a Mechanics Explorer visualization pane and zoom in or out using the mouse scroll wheel.
  6. Change the controller setpoint using the reference dial to elicit a step response.
  7. View the response characteristics in the dashboard scope.
  8. Adjust the controller parameters and continue interacting.
  9. When you are finished, stop the model ■.

Model structure

Each model has three main components:
  1. An interaction layer at the top level of the model. This includes dashboard blocks to interact with the controller and scopes to view essential signals.
  2. The Simulink model. This is also included in the top level of the model, just below the dashboards. The Simulink model implements the controller. The plant is also present in the Simulink model as a masked subsystem.
  3. The virtual mechanism. The mechanism is implemented using Simscape Multibody. You can double-click the mask of the virtual mechanism to adjust the physical parameters, such as the pendulum length.
